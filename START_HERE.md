# 🎉 Radhe Boutique - START HERE

Welcome to your production-ready luxury jewellery e-commerce platform!

## ⚡ Quick Start (5 Minutes)

```bash
# 1. Install dependencies
npm install

# 2. Create environment file
cp .env.example .env

# 3. Add minimum MongoDB URI to .env
MONGODB_URI=mongodb://localhost:27017/radhe-boutique

# 4. Seed database with sample data
npm run seed

# 5. Start development server
npm run dev

# 6. Open browser → http://localhost:3000
```
